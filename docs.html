Hordamy Ensino Descentralizado e Livre


1.Introdução

AHordamyé uma plataforma inovadora e tecnológica.Com um grande diferencial que reside no uso de padrões de smart contract,que funcionam como acesso e propriedade dos produtos do seu marketplace.Ao adquirir um acesso na Hordamy,o usuário recebe um token não fungível exclusivo que garante a propriedade de acesso ao conteúdo.

Com este modelo descentralizado,a plataforma oferece maior segurança e flexibilidade,permitindo que os usuários,transfiram ou revenda seus acessos quando quiser.

2.Estrutura Técnica

A Hordamy foi construída sobre uma arquitetura,que integra tecnologias blockchain e ferramentas web modernas e tradicionais para fornecer uma experiência de usuário eficiente e dinâmica.Abaixo está uma visão geral da estrutura técnica da plataforma:

  • Front-end:O front-end é desenvolvido utilizando linguagens e padrões tradicionais da web2,criando uma interface responsiva e intuitiva.
  • Blockchain:Utilizando redes descentralizadas e criptografadas como Polygon,filecoin,IPFS...Para gerenciar transações por meio de contratos inteligentes.
  • Tokens ERC20:Cada produto é associado a um token de acesso específico,que garante assim a propriedade,descentralização e acesso ao conteúdo.
  • MetaMask:A integração permite que os usuários autentiquem login na plataforma e interaja com os seus conteúdos,sem necessidades de cadastros,dados sensíveis,autoridades e etc.

A verificação de propriedade do acesso é realizada automaticamente através de verificação senha de acesso único vinculado a um padrão tokenizado,garantindo que o usuário só terá acesso se realmente possuir o contrato necessário necessário.

3.Integração com Polygon

A Polygon é uma extensão essencial para a plataforma Hordamy,permitindo a interação direta dos usuários com a blockchain.Através de sua wallet de preferência,os usuários podem gerenciar suas tokens digitais,acesso e realizar transferências seguras.

3.1 Requisitos

  • Wallet.
  • Conexão com a internet.
  • Possuir o acesso referente ao produto desejado.

3.2 Processo de Autenticação

  1. O usuário acessa a plataforma e seleciona o produto desejado.
  2. É solicitado o login e senha para acessar o conteúdo desejado.
  3. O sistema verifica se o usuário possui permissão correspondente ao seu produto.
  4. Se o token de acesso estiver presente,o produto é desbloqueado para o usuário usufruir.

4.Token de acesso

O token de acesso é a base para a concessão aos produtos na Hordamy.Cada conteúdo possui um token de acesso caracterizado como especial,exclusivo,gold e diamante,que funciona como uma "chave" ao conteúdo.Este modelo permite que o usuário seja o verdadeiro proprietário de acesso.

4.1 Padrão ERC20

O padrão ERC define o conjunto de regras que permite a criação de smart contract na blockchain.Esse padrão pode ser transferido ou armazenado em qualquer carteira compatível.

4.2 Benefícios do Modelo de acesso

  • Propriedade:O usuário é o verdadeiro proprietário do acesso,representado pelo seu token.
  • Transferência:O usuário pode transferir ou revender o seu acesso para quem quiser inclusive dentro dos mercados de NFT,permitindo que outros usuários tenham também acesso ao produto.
  • Segurança:A propriedade e as transações são registradas na blockchain da polygon,garantindo a transparência,autenticidade e integridade da transação.
